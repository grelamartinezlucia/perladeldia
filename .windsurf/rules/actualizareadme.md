# Regla: Actualizar README

Cada vez que se añadan funcionalidades nuevas al bot, se debe actualizar el archivo README.md para reflejar:

- Nuevos comandos añadidos
- Cambios en comandos existentes
- Nuevas características o comportamientos
- Comandos de administrador nuevos

El README debe mantenerse siempre actualizado con la documentación del proyecto.
